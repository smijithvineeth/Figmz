================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "python3.12 not found"
Solution: Install Python 3.12 from https://python.org

Problem: "No module named 'fastapi'"
Solution: Make sure virtual environment is activated:
          venv\Scripts\activate

Problem: Installation takes too long
Solution: This is normal! dlib compiles from source. Takes 5-10 minutes.

Problem: Camera not working
Solution:
   1. Grant browser permission to camera
   2. Check camera isn't used by another app
   3. Refresh the page

Problem: Low recognition accuracy
Solution: Capture more photos (10+ per person) with different angles
